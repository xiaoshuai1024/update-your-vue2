# v0.2 Implementation Plan (update-your-vue2)

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Implement v0.2 capabilities: apply codemods that actually modify code (JS/TS + Vue SFC), analyze and update `package.json` scripts following `--target`, add `--install` support, enhance `restore` with pre-backup, and deepen build migration for `vite` and `webpack`.

**Architecture:** Keep a single “plan → ChangeQueue → apply” pipeline. Every step contributes changes into `src/changes/ChangeQueue`, enabling `--dry-run` preview and deterministic application. Codemods produce file-level edits; scripts migration produces `package.json` edits with backups; install is optional and runs after changes apply.

**Tech Stack:** Node.js + TypeScript, Vitest, `fast-glob`, `ignore`, `archiver`, `adm-zip`, (add) `@vue/compiler-sfc`, optional AST tooling (`recast`/`jscodeshift`) only when needed.
